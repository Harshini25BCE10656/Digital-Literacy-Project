Project Overview 
This repository contains my Digital Literacy Portfolio, submitted as part of the CSE0001 course at VIT Bhopal University. As a Student Digital Ambassador, I completed five tasks covering digital tools, professional online presence, coding platforms, email etiquette, and cybercrime awareness. 

Module Summaries
Task 1 – Digital Literacy Infographic 
Created a one-page infographic using Canva covering what digital literacy is, safe internet practices, and professional online presence. 
https://github.com/Harshini25BCE10656/Digital-Literacy-Project/tree/main/task-1-presentation

https://github.com/Harshini25BCE10656/Digital-Literacy-Project/tree/main/task-2-portfolio
